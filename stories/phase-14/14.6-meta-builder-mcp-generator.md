# Story 14.6 — Meta-Builder MCP Skill Generator

## Phase
Phase 14: Visual Compilation Engine

## Dependencies
- Story 13.3, 14.2 must be `DONE`.

## Status
`[ ] PENDING`

## Description
Implement a Meta-Agent that generates Node.js skill execution logic and valid JSON Schemas from natural language requests, routing the generated code into the Monaco Editor for review before saving.

## Acceptance Criteria
- [ ] "Generate Skill" section in the Skills & MCPs tab.
- [ ] Textarea: "Describe what this skill should do (e.g., 'Query our internal Postgres database with a SQL string and return rows as JSON')."
- [ ] "Generate" button calls `POST /api/workspaces/:id/skills/generate`.
  - Request: `{ description: string }`.
  - Backend Meta-Agent prompt: generate both a TypeScript handler function AND a JSON Schema for the skill's input parameters.
  - Response: `{ handlerCode: string; jsonSchema: object; skillName: string; description: string }`.
- [ ] Generated handler code displayed in a Monaco Editor (TypeScript mode, editable).
- [ ] Generated JSON Schema displayed in a second Monaco Editor (JSON mode, editable).
- [ ] "Save Skill" button: compiles the skill (story 14.2) and saves to workspace storage (story 14.3).
- [ ] Saved skill registered in the workspace MCP registry and available immediately.
