# Story 14.2 — Abstract File Templating Engine

## Phase
Phase 14: Visual Compilation Engine

## Dependencies
- Story 5.1, 14.1 must be `DONE`.

## Status
`[ ] PENDING`

## Description
Create an Express utility that accepts JSON payloads from the frontend builders and compiles them into valid `.md` (Agent) and `.js` (Skill) files.

## Acceptance Criteria
- [ ] `FileTemplatingEngine.compileAgent(personaData: AgentPersonaForm): string` — produces a valid agent `.md` file with YAML frontmatter and system prompt body.
- [ ] `FileTemplatingEngine.compileSkill(skillData: SkillForm): { config: string; handler: string }` — produces `skill.config.json` and a TypeScript handler stub.
- [ ] `POST /api/workspaces/:id/compile/agent` — compile and preview the agent `.md` output (does not save to disk).
- [ ] `POST /api/workspaces/:id/compile/skill` — compile and preview the skill files.
- [ ] "Preview" button in AgentsTab shows the compiled output in a Monaco read-only editor.
- [ ] Validation: compiled `.md` must have valid YAML frontmatter (parsed and verified before returning).
- [ ] Compiled skill handler TypeScript must be syntactically valid (checked with `ts-morph`).
