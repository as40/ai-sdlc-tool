## Description
Build a backend `CryptoService` using Node.js's built-in `crypto` module with AES-256-GCM authenticated encryption. This service will be used by all subsequent stories that need to store sensitive strings (Git tokens, LLM API keys) securely in the database.

## Acceptance Criteria
- [ ] `CryptoService` class with two public methods: `encrypt(plaintext: string): string` and `decrypt(ciphertext: string): string`.
- [ ] Encryption algorithm: **AES-256-GCM** (authenticated encryption — prevents tampering).
- [ ] A unique random 16-byte IV is generated for every `encrypt()` call.
- [ ] The output format of `encrypt()` is a single base64 string encoding: `IV (16 bytes) + Auth Tag (16 bytes) + Ciphertext`.
- [ ] `decrypt()` parses this format, verifies the auth tag, and returns the original plaintext.
- [ ] Encryption key sourced exclusively from `ENCRYPTION_KEY` env var (must be 64 hex characters = 32 bytes).
- [ ] Service throws a typed `CryptoError` (extends `AppError`) if key is missing, malformed, or auth tag verification fails.
- [ ] `CryptoService` is a singleton instantiated once at app startup via DI.
- [ ] Unit tests achieve 100% coverage on the service.

## Technical Notes
- Use `crypto.createCipheriv('aes-256-gcm', key, iv)` — never `createCipher` (deprecated).
- Key must be a `Buffer` of exactly 32 bytes derived from the hex string: `Buffer.from(process.env.ENCRYPTION_KEY, 'hex')`.
- Auth tag retrieved via `cipher.getAuthTag()` after `cipher.final()`.
- Never log the plaintext or the key.
- The service is pure (no DB, no HTTP) — easy to unit test without mocks.

## Tests to Write (`crypto.service.test.ts`)
```typescript
describe('CryptoService', () => {
  describe('encrypt', () => {
    it('should return a non-empty base64 string')
    it('should produce different ciphertexts for the same plaintext (unique IV)')
    it('should throw CryptoError when ENCRYPTION_KEY is missing')
  })
  describe('decrypt', () => {
    it('should return the original plaintext after encrypt → decrypt round-trip')
    it('should throw CryptoError if ciphertext is tampered with')
    it('should throw CryptoError if auth tag is invalid')
    it('should throw CryptoError for malformed input')
  })
})
```
