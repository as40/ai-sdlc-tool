## Description
Integrate Figma design reading via MCP and build an image upload pipeline that converts screenshots/designs to Base64 for Claude's vision API.

## Acceptance Criteria
- [ ] **Figma MCP Integration**:
  - Register Figma MCP server in the skill registry.
  - Skill: `read_figma_component(figmaUrl)` — calls Figma API to retrieve component data as JSON.
  - API key for Figma stored encrypted in workspace config.
- [ ] **Image Upload Pipeline**:
  - Chat input supports image paste (Ctrl+V) and drag-drop.
  - `POST /api/workspaces/:id/uploads/image` (multipart): accepts PNG/JPG up to 5MB.
  - Validates MIME type server-side (not just extension).
  - Converts image to Base64.
  - Returns `{ base64: string; mediaType: string }` — stored temporarily in session context.
  - When chat message sent, images are included as vision blocks in the Claude API call.
- [ ] Image thumbnails shown as attachments above the chat input before sending.
- [ ] Remove attachment button on each thumbnail.
- [ ] Max 3 images per message.
