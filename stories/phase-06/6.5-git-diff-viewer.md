# Story 6.5 — Right Sidebar Live Git Diff Viewer

## Phase
Phase 6: Visual Workspace IDE & Live Code Editor

## Dependencies
- Story 6.1, 6.2 must be `DONE`.

## Status
`[ ] PENDING`

## Description
Construct a dedicated Git diff viewer in the Right Sidebar showing all unstaged changes, with a split-pane comparison view when a file is selected.

## Acceptance Criteria
- [ ] `GET /api/workspaces/:id/git/diff` — returns list of changed files and their diff stats.
  - Response: `[{ path, additions, deletions, status: 'modified'|'added'|'deleted' }]`.
- [ ] `GET /api/workspaces/:id/git/diff/:encodedPath` — returns the full unified diff for one file.
- [ ] Right Sidebar shows the changed files list with +/- counters and status icons.
- [ ] Clicking a file opens a split-pane Monaco diff editor (side-by-side: original vs. modified).
- [ ] Diff viewer auto-refreshes every 5 seconds or when a `file_saved` WebSocket event is received.
- [ ] "Stage All" button: runs `git add -A` for the workspace repo.
- [ ] "Discard All" button: runs `git checkout -- .` (with confirmation dialog).
- [ ] Summary line at top of sidebar: "X files changed, +Y insertions, -Z deletions".
