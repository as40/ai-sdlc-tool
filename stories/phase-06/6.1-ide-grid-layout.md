# Story 6.1 — Main IDE View Grid Layout

## Phase

Phase 6: Visual Workspace IDE & Live Code Editor

## Dependencies

- Story 1.1 (frontend scaffold) must be `DONE`.

## Status

`[ ] PENDING`

## Description

Construct the three-panel workspace IDE layout: Left Sidebar, Center Display Area, and Right Sidebar using resizable panels.

## Acceptance Criteria

- [ ] Main IDE page at route `/workspace/:workspaceId/ide`.
- [ ] Three-panel layout using `react-resizable-panels` (or equivalent).
  - **Left Sidebar** (default 240px, min 160px): File Explorer + Jira panel tabs.
  - **Center Area** (flex-grow): Code Editor / Chat view toggle.
  - **Right Sidebar** (default 320px, min 200px): Git Diff viewer.
- [ ] Panels are resizable by dragging the dividers.
- [ ] Panel sizes persisted to `localStorage` across sessions.
- [ ] Left and Right sidebars can be collapsed/expanded with a toggle button.
- [ ] Top navigation bar with: workspace name, branch dropdown (placeholder), execution mode toggle (placeholder), user avatar + role.
- [ ] Responsive design: on screens < 1024px, sidebars default to collapsed.
- [ ] Dark theme by default (Tailwind dark mode classes).
- [ ] Route `/workspaces/:id/ide` added to `App.tsx`. Requires `ProtectedRoute` with minimum `DEVELOPER` role.
- [ ] Navigation flow: from `/workspaces/:id/overview` (Story 4.3), a 'Open IDE' or 'Start Coding' button navigates to `/workspaces/:id/ide`.
- [ ] TopNavBar shows a loading skeleton while workspace data is being fetched (use shadcn `Skeleton` class: `animate-pulse rounded bg-zinc-800`).
